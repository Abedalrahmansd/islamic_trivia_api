Complete Islamic Trivia Game API

A complete, bilingual (English/Arabic) REST API for an Islamic Trivia Game supporting both web and mobile platforms. This API provides comprehensive game management, content administration, and analytics capabilities.

Version 1.0.0 PHP 7.4+ MySQL 5.7+ MIT License
Get Started API Reference

API Overview

📊

Complete CRUD Operations

Manage all aspects of your trivia game with comprehensive CRUD operations for:

  • Categories with difficulty levels and timers
  • Challenge packs with themes and download tracking
  • Questions with bilingual content and explanations
  • Game sessions with team management
🔐

Admin Authentication & Authorization

Secure admin access with JWT-based authentication and role management:

  • JWT token authentication for all admin endpoints
  • Role-based access control (admin, super_admin)
  • Comprehensive admin action logging
  • Profile management with password updates
🌍

Bilingual Support

Full English/Arabic content support throughout the API:

  • All text fields have English and Arabic versions
  • Bilingual category names, descriptions, and content
  • Arabic question text and answer options
  • Explanations in both languages
🎮

Game Session Management

Complete game flow from creation to results tracking:

  • Create game sessions with custom parameters
  • Support for multiple teams and rounds
  • Save detailed game results with per-question analytics
  • Retrieve completed game details and statistics
📈

Statistics & Analytics

Comprehensive dashboard and reporting capabilities:

  • Dashboard statistics with counts and recent activity
  • Category, pack, and question statistics
  • Game performance analytics
  • General system statistics
🤖

AI Content Generation

AI-powered question and content generation (integration ready):

  • Generate content using AI models like GPT-3.5
  • Flexible API for integrating with various AI services
  • Track AI usage and costs
  • Content suggestion system

Installation & Setup

Prerequisites

  • PHP 7.4 or higher - Required for running the API backend
  • MySQL 5.7 or higher - Database for storing trivia content and game data
  • Web server - Apache or Nginx to serve the API endpoints
  • Composer - PHP dependency manager
  • Git - For cloning the repository

Base URL & Authentication

BASE URL: https://yourdomain.com/api/
Authentication: Admin endpoints require JWT token in Authorization header:
Authorization: Bearer {your_jwt_token}

Standard Installation

git clone https://github.com/Abedalrahmansd/islamic_trivia_api.git cd islamic_trivia_api

Database Setup

mysql -u yourusername -p yourdatabase < database.sql

Environment Configuration

cp .env.example .env # Edit .env with your configuration

Docker Installation (Alternative)

docker-compose up -d docker exec -it trivia-app composer install docker exec -it trivia-app php database/seed.php

Security Checklist

  • Change default database credentials
  • Set strong JWT secret key
  • Implement HTTPS in production
  • Configure rate limiting
  • Set up proper error logging
  • Configure firewall rules
  • Enable regular security updates
  • Use strong admin passwords
  • Implement database backups
  • Monitor access logs regularly
  • Validate and sanitize all inputs

Database Schema

Main Tables

The Islamic Trivia API uses a relational database structure with the following main tables:

categories

Stores question categories with difficulty levels, timers, and bilingual names.

challenge_packs

Stores downloadable question packs with themes and download tracking.

questions

Stores questions with bilingual content, options, explanations, and metadata.

games

Stores game session records with configuration and completion status.

game_teams

Stores teams participating in games with final scores and rankings.

game_results

Stores individual question results with detailed analytics.

admins

Stores administrator accounts with roles and authentication data.

admin_logs

Stores admin action audit trail for security and monitoring.
